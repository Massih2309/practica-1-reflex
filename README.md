# 🚀 Actividad 5 — Landing Page Moderna con Reflex

Bienvenido a este proyecto desarrollado con **Reflex (Python)**.  
El objetivo de esta práctica es demostrar cómo crear una **Landing Page profesional, moderna y responsive** utilizando únicamente Python para el frontend y backend.

Este proyecto está enfocado en aplicar buenas prácticas de desarrollo web, diseño limpio y organización estructurada de componentes.


---

# 🛠️ Tecnologías Utilizadas

- **Lenguaje:** Python 3.x  
- **Framework:** Reflex  
- **Control de versiones:** Git & GitHub  
- **Entorno virtual:** venv 

---

# 🚀 Instalación y Ejecución

## 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
```

## 2️⃣ Entrar al proyecto

```bash
cd TU-REPOSITORIO
```

## 3️⃣ Crear entorno virtual (Opcional pero recomendado)

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 4️⃣ Instalar dependencias

```bash
pip install reflex
```

o

```bash
pip install -r requirements.txt
```

---

## 5️⃣ Ejecutar el proyecto

```bash
reflex run
```
